# dev-atlas

> A navigational reference for software engineering — concepts, patterns, tools, and more.

This is an open-source knowledge base written in Markdown. Think of it as a personal wiki: structured, linkable, and built gradually over time.

---

## How to navigate

Each section has its own `README.md` that lists its articles. Use GitHub's file browser or the links below to explore.

---

## Article structure

Every article follows the same template:

- **What is it?** — a clear definition
- **Why does it matter?** — the motivation
- **How it works** — the mechanism
- **Examples** — concrete illustrations
- **When to use / When NOT to use** — trade-offs
- **References** — sources and further reading
